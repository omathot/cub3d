# cub3d

This project is a simple raycasting game developed for the 42 curriculum, compatible with Linux and MacOS.

- [sboulain](https://github.com/P1tayaa)

## Installation

1. Clone the repository:
   ```bash
   git clone <repository_url>
   cd cub3d
   make test
   ```
2. Clean Up:
   ```bash
    make hardclean
    ```
